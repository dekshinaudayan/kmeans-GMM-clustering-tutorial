# K-Means vs Gaussian Mixture Models — Clustering Tutorial

This repository contains the code, figures, and tutorial for a comparison between K-Means and Gaussian Mixture Models (GMM) as part of a machine learning coursework assignment.

# Contents

kmeans_gmm_demo.ipynb — Jupyter notebook with all code and plots
report.pdf — Final written tutorial (<2000 words)
LICENSE — MIT License

# Techniques Covered

K-Means clustering
Gaussian Mixture Models
Elbow method
Silhouette score
AIC/BIC model selection
Failure case on anisotropic data

# How to Run the Notebook

1.Install Python 3.8+
2.Install dependencies:
    CODE : pip install numpy matplotlib scikit-learn
3. Open the notebook
    CODE:jupyter notebook kmeans_gmm.ipynb
